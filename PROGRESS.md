# わかたけショップ 開発進捗

## 構成ファイル

| ファイル | 役割 |
|---|---|
| `my-dept-export.php` | WordPress/WooCommerce プラグイン（CSV出力） |
| `csv_to_excel.py` | 部署金額まとめ表 生成スクリプト |
| `delivery_note.py` | 納品書 生成スクリプト |
| `menu.py` | 管理ツール ランチャーメニュー |
| `テンプレート/部署金額まとめ表.xlsx` | 部署別Excel テンプレート（書式参照用） |
| `テンプレート/ダイキン用納品書（総まとめ）.xlsx` | 納品書テンプレート（流し込み用） |
| `dist/管理ツール.exe` | ビルド済みexe |

---

## WordPress プラグイン（my-dept-export.php）

**バージョン:** 1.7

### 機能

#### 部署別に出力
- `billing_company`（請求先会社名）で注文を絞り込み
- 「全て」チェックボックスで一括選択/解除
- CSV列：注文番号・注文日・部署名・氏名・カテゴリ・商品内容・合計金額(税込)

#### 商品カテゴリ別に出力
- 商品カテゴリのスラッグで注文を絞り込み
- 「全て」チェックボックスで一括選択/解除
- CSV列：同上

#### 納品書用に出力
- ボタン1つで全注文を商品名ごとに集計して出力（選択不要）
- CSV列：商品名・単価(税込)・合計数量・合計金額(税込)

### 共通仕様
- 文字コード：UTF-8 + BOM（Excelで開いても文字化けしない）
- 対象ステータス：処理中（wc-processing）・完了（wc-completed）

## 管理ツール exe（menu.py → dist/管理ツール.exe）

ダブルクリックで起動するGUIランチャー。ツールを追加するたびに`menu.py`の`TOOLS`リストに追記してexeを再ビルドする。

### ツール一覧

#### 1. 部署金額まとめ表を作成（csv_to_excel.py）
- CSVファイルを選択するとExcelを自動生成・オープン
- 出力：`部署金額まとめ表_日時.xlsx`（CSVと同じフォルダ）
- 3シート構成：
  - 集金用封筒（各部署へ配布）― 部署名・氏名・金額
  - 仕分け時ダンボール貼り付け用 ― 部署名・氏名のみ
  - 組合事務所での集金用 ― 部署名・氏名・金額
- 書式：游ゴシック 16pt・A4横・5列グリッド・金額は全角

#### 2. 納品書を作成（delivery_note.py）
- 「納品書用CSV」を選択するとテンプレートに流し込んでExcelを自動生成・オープン
- テンプレート：`テンプレート/ダイキン用納品書（総まとめ）.xlsx`（exeと同じ階層に配置）
- 出力：`納品書_日時.xlsx`（CSVと同じフォルダ）
- 納品日を実行日の日付に自動更新
- データ行（7〜167行目）に商品名・単価・数量・合計式を書き込む

### exeのビルド方法
```
cd wakatake-shop
pyinstaller --onefile --windowed --name "管理ツール" menu.py
```

### 配布時のフォルダ構成
```
任意のフォルダ/
├── 管理ツール.exe
└── テンプレート/
    ├── ダイキン用納品書（総まとめ）.xlsx
    └── 部署金額まとめ表.xlsx
```

---

## 既知の注意点

- PHP側のCSVはUTF-8 BOMで出力しているが、Pythonで読む際にBOMがヘッダーキーに残ることがある → `lstrip('﻿')` で対処済み
- 納品書テンプレートのデータ行は最大161行（7〜167行目）。それ以上の商品数には対応していない
